Enumeration Percent​Decode​Set 

public enum PercentDecodeSet

A namespace for percent-decode sets.

Decoding is thankfully much simpler than encoding; in almost all cases, simply removing percent-encoding is sufficient, as regular URL encode-sets do not substitute characters. Form-encoding is the only exception specified in the URL Standard.

Since percent-decode sets are not stateful, you only ever need to refer to their type, never an instance. The types are exposed as properties so you can use a convenient KeyPath syntax to refer to them.
