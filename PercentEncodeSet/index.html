Enumeration Percent​Encode​Set 

public enum PercentEncodeSet

A namespace for percent-encode sets.

Since percent-encode sets are not stateful, you only ever need to refer to their type, never an instance. The types are exposed as properties so you can use a convenient KeyPath syntax to refer to them.
