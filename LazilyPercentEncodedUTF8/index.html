Typealias Lazily​Percent​Encoded​UTF8 

public typealias LazilyPercentEncodedUTF8<Source, EncodeSet> =
FlattenSequence<LazilyPercentEncodedGroups<Source, EncodeSet>>
where Source: Collection, Source.Element == UInt8, EncodeSet: PercentEncodeSetProtocol

A Collection which lazily percent-encodes its Source UTF8 code-units using a given EncodeSet. This collection only adds percent-encoding or substitutions; it does not decode any pre-existing percent-encoded or substituted code-points in Source.

Percent encoding transforms arbitrary Unicode strings to a limited set of ASCII code-points which are permitted by the EncodeSet. If the EncodeSet performs substitutions, users should take care to decode the contents using the same EncodeSet.
